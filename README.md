# amazon-review-sentiment-analysis
Amazon Review Sentiment Analysis using NLP and Machine Learning (TF-IDF + Logistic Regression)

# Amazon Sentiment Analysis
## 📌 Project Overview
This project focuses on analyzing Amazon product reviews and classifying them into positive or negative sentiments using Natural Language Processing (NLP) techniques.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP (TF-IDF Vectorization)
- Matplotlib & Seaborn

## ⚙️ Project Workflow
1. Data loading from compressed (.bz2) files
2. Data preprocessing and text cleaning
3. Label extraction (positive/negative)
4. Feature extraction using TF-IDF
5. Model building using Logistic Regression
6. Model evaluation using accuracy and classification report

## 📊 Results
- Accuracy: ~89%
- Balanced precision and recall for both classes

## 🎯 Key Features
- Handled large compressed dataset
- Applied text preprocessing techniques
- Built a machine learning model for sentiment classification

## 🚀 Conclusion
The model successfully predicts whether a review is positive or negative and can help businesses understand customer feedback.
